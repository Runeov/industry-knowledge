# Industry Knowledge — Sprint 1 Evidence Pack

This folder contains the documentation evidence for **Sprint 1** of the
Industry Knowledge (FM2AJIK05) module at Noroff Vocational College
(Frontend Year 2). The sprint covered building a simple responsive
one-page site (hero / about / contact) with a Netlify CMS admin
relocated under `/admin/`.

For project setup and deployment, see the root [README](../README.md).

This is a **4-person practice run**. Rune (`Runeov`) is the real
student and Scrum Master. **Member B, Member C, and Member D are
simulated** so that the full sprint process — planning, daily
standups, blocker handling, QA loop, retrospective — can be
demonstrated end-to-end. All decisions, communications, and outcomes
recorded here reflect work actually performed by Rune playing through
the roles for the practice simulation.
